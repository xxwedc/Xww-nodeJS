# Xww-nodeJS
基于Node.js的校园智慧小程序系统。包含寻物启事、二手物品出售等模块

## **使用说明：**

serve为服务器文件目录 (运行环境需要安装node.js)

1、下载node_modules依赖。管理员状态下运行cmd。安装命令如下：

```
npm install 
```

如果使用yarn包管理工具，则命令如下：

```
yarn install
```

2、数据库连接，在serve/config/index.js文件中配置自己的数据库连接信息和密码。

3、运行app.js文件，命令行如下：

```
node app.js
```
